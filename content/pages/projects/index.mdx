## [Beat My Rate - Loan Officer Marketplace](https://frontend-production-0858.up.railway.app/)

Full-stack bidding platform automating mortgage rate competition between loan officers. Borrowers upload financial documents which undergo automated parsing and data extraction, triggering real-time bidding among pre-qualified officers filtered by loan type and geography. WebSocket implementation enables live bid updates, while machine learning algorithms optimize officer-borrower matching based on historical success rates and specialization areas. Streamlines traditionally manual loan shopping process into efficient marketplace dynamics.

## Eventify.ai - AI Event Planning Platform

Autonomous event planning system using AI voice agents to negotiate with vendors via phone calls. Platform processes event requirements then deploys Python agents with Bland AI to contact restaurants, venues, and decorators for real-time price negotiation. Features automated quote extraction, email scraping, and comparison dashboard built with React, Node.js, Supabase, and Google Gemini for processing vendor responses.

## Homeyfyi.tech - AI Home Inspection Platform

Computer vision system automating property inspections through image/video analysis to identify structural issues and predict maintenance needs. ML models evaluate foundation integrity, roof condition, and environmental factors, generating comprehensive health scores and 3-6 month risk predictions. Integrates with insurance APIs for premium calculations based on assessed property conditions and location-specific hazards.

Flutter mobile app calculating personalized caloric requirements and macronutrient distributions based on user profiles. Integrates Spoonacular API for nutrition data, Firebase for authentication and data persistence, enabling custom meal planning and shopping list generation. Features social sharing, collaborative shopping lists, and favorite meal saving for comprehensive nutrition management.
